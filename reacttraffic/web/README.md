## 專案資訊

- 前端框架: React@17.0.1
- UI 套件:
  - PrimeReact@6.5.0
  - PrimeIcons@4.1.0
  - PrimeFlex@2.0.0
- 圖表套件:
  - eCharts@5.1.2

## 建置步驟

- 複製專案中的.env.example 並更名為.env

```
GENERATE_SOURCEMAP=false
REACT_APP_ENV=development
PORT=8000
```

- 至以下網站下載 12.10.0 版本

```
https://nodejs.org/en/download/releases/
```

### 安裝相對應 Node 版本

- 至以下網站下載 12.10.0 版本

```
https://nodejs.org/en/download/releases/
```

### 安裝套件

- 開啟終端機，依以下步驟建置
- 安裝套件

```
npm install
```

- 開啟服務

```
npm start
```

- 打包專案

```
npm build
```
